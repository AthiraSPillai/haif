# HAIF Python

Python CLI and SDK for the Human-Agent Intent Framework.

The Python package mirrors the core HAIF operations for AI engineering, data, and enterprise automation teams:

```bash
python -m haif.cli init
python -m haif.cli new proposal "Reduce duplicate agent-created tickets"
python -m haif.cli validate
python -m haif.cli preflight --scope jira,docs
python -m haif.cli export-context
```

It is dependency-free and works with Python 3.8+.
